Setup

What's your ETHZ username?

Before enabling ETHZ Auto-Login, save your ETHZ password in your browser's password manager. The extension uses the password that the browser fills on the login page.

Save your password in browser passwords:
  1. Open your browser password manager.
  2. Add or update the login for aai-logon.ethz.ch.
  3. Save your nethz username and ETHZ password there.

Press Enter ↵

Browser passwords

Save it in your password manager.

Save your ETHZ password in your browser passwords first. The extension uses the password your browser password manager fills, then continues the login automatically.

Use aai-logon.ethz.ch as the saved login site. If browser password manager autofill is disabled, you can optionally store the password in this extension from the popup.
